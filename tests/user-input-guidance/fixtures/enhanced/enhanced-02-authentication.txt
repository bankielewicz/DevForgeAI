Build login system for customers and end users using JWT.
Customers and end users can sign up, log in, reset passwords.

Passwords: 12+ chars with upper, lower, number, special char.
Login within 200ms. Use JWT tokens (24 hour validity).

Given: End user enters signup info.
When: System checks email and password.
Then: Account created with email verify (24 hour expiry).

Given: End user logs in.
When: Email and password correct.
Then: Login in 200ms, JWT token generated.

Given: End user forgets password.
When: Reset requested.
Then: Reset link sent (1 hour expiry).

Security: Bcrypt password hash. TLS 1.2+ encryption.
Log failed logins. Block after 5 attempts.
