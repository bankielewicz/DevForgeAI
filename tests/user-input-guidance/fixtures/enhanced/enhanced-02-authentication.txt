Design authentication system for user registration, login, and password recovery. Users register with email/password (12+ chars, uppercase, lowercase, number, special char), authenticate via JWT within 200ms, and recover forgotten passwords via 1-hour links. Given: User enters valid signup credentials. When: System validates email and password requirements. Then: Account created with verification email (24 hour expiration). Given: Registered user logs in. When: User enters correct email and password. Then: System authenticates within 200ms and creates JWT session (24 hour validity). Given: User requests password reset. When: User clicks reset link (1 hour expiration). Then: System allows password change with same requirements. Security Requirements: bcrypt
