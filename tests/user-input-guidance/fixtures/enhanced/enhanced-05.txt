Feature: User Profile Management with Audit Trail

Enable users to manage their profile with comprehensive versioning and audit capabilities.

Acceptance Criteria:
1. Users edit name, email, phone number, and address information
2. All changes validated against business rules and data formats
3. Profile picture upload supports JPEG, PNG up to 5MB
4. Changes saved atomically to database with version numbers
5. Complete audit trail maintained: who changed what and when
6. Users view change history showing before/after values for each field
7. Email changes require confirmation via verification link
8. Phone number validated using international format checking

Non-Functional Requirements:
- Profile updates respond within 300ms
- Profile picture upload and resize completes in <2 seconds
- Audit trail retained for 7 years for compliance
- Support 10,000 concurrent profile edits
- Profile image stored in CDN for fast retrieval

Technical Specification:
- Tables: users (id, name, email, phone, address), user_audit (id, user_id, field, old_value, new_value, changed_by, changed_at)
- Image storage: S3 with CloudFront CDN, auto-resize to multiple dimensions
- API endpoints: PUT /api/profile, GET /api/profile/history, POST /api/profile/picture
- Validation: email RFC 5322, phone E.164 format, address geocoding
