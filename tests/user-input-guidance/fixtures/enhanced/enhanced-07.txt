Feature: Analytics Dashboard with Multi-Dimensional Reporting

Create comprehensive dashboard for business metrics with filtering and export capabilities.

Acceptance Criteria:
1. Revenue metrics: total sales, daily revenue trends, revenue by product, by region
2. Order metrics: total orders, average order value, orders by status, repeat customer rate
3. Customer metrics: total active customers, new customer growth, customer lifetime value
4. Product metrics: top 10 products by revenue, products by category, inventory levels
5. Regional analysis: sales by geography, customer distribution, shipping costs by region
6. Date filtering: date range, preset filters (today, this week, this month, this year)
7. Export functionality: download reports as CSV or PDF
8. Dashboard loads in under 3 seconds with cached aggregated data

Non-Functional Requirements:
- Report generation completes in <5 seconds
- Dashboard supports 1000+ concurrent views
- Data updated hourly (batch processing at night)
- Archive reports for historical comparison
- Role-based access: managers see all regions, regional managers see only their region

Technical Specification:
- Database: PostgreSQL with materialized views for aggregated metrics
- Batch jobs: nightly aggregation of sales, order, customer data
- Caching: Redis cache for dashboard metrics with 1-hour TTL
- Frontend: React with Chart.js for visualizations
- API endpoints: GET /api/analytics/revenue, /api/analytics/orders, /api/analytics/customers
