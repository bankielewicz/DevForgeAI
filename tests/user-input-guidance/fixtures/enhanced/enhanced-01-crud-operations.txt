User account system for admin users. Must support 100,000 active users.
CRUD under 500ms. Bulk 100 users under 5 seconds.

Admin users create, read, update, delete accounts.
Each account needs email and role.

Given: Admin creates account with email and role.
When: Creation completes.
Then: Confirm in under 500ms.

Given: Account exists.
When: Admin updates email or role.
Then: Changes save in under 500ms.

Given: Account exists.
When: Admin deletes account.
Then: Archive for 90 days.

Security: Enforce role checks and audit logging with timestamp and admin ID.
Use TLS 1.2+ encryption.

Performance and scalability: Handle 50 admin users concurrently with no slowdown.
