Implement a user account management system specifically for system administrators providing CRUD operations on user accounts. Support 100,000 users with response times under 500ms for all operations.

Acceptance Criteria:
- Given: Administrator creates new user with email and assigned role
  When: User creation completes
  Then: System confirms within 500ms

- Given: User account exists in system
  When: Administrator updates email or role
  Then: Changes persist immediately

- Given: User record exists
  When: Administrator deletes account
  Then: Account archived for 90 days

Non-Functional Requirements:
- Performance: CRUD operations <500ms p95; bulk operations (100+ users) <5 seconds
- Security: Role verification required; changes logged with timestamp and operator ID; TLS 1.2+ encryption
- Scalability: Support 50 concurrent administrators
