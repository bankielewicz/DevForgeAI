Stripe API v3 integration for customer and client payments.
Requirement: 3 second response timeout maximum.
Constraint: Handle success, failure, decline, timeout cases.

When: Payment system calls Stripe with token.
Then: Payment in 3 seconds with invoice.

Given: Stripe fails.
When: First attempt fails.
Then: Retry 3 times (1s, 2s, 4s backoff).

Given: Stripe webhook arrives.
When: Webhook received.
Then: Verify HMAC-SHA256 signature.

Valid: update plan, email in 5s.

Handle 25 Stripe error codes. Store transaction records via tokens only, never card data.
Security: PCI DSS Level 1, HTTPS/TLS 1.2+, env keys.
