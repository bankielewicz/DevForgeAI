# Story: When to Hire a Professional Framework

## Description

**As a** entrepreneur or small business owner seeking legal guidance,
**I want** a structured decision framework that identifies when my situation requires professional legal counsel rather than self-help resources,
**so that** I can make informed decisions about when to invest in attorney services, avoid costly legal mistakes from under-preparation, and approach professional consultations with confidence.

## Non-Functional Requirements (NFRs)

### Performance

**Response Time:**
- Reference file load and first content render: < 200ms (p95)
- Complexity indicator evaluation: < 50ms per pass (p99)
- File size: under 80KB

### Security

**Data Protection:**
- No user input persisted or logged with PII
- Reference file read-only at runtime
- Attorney referral URLs static and curated (no dynamic resolution)

### Scalability

**Design:**
- Supports addition of up to 20 new complexity indicators without structural refactoring
- Progressive disclosure supports 5+ additional resource categories within 999-line constraint
- Stateless per invocation

### Reliability

**Error Handling:**
- Valid response for 100% of well-formed inputs (referral or self-help)
- Ambiguous input triggers clarification request, not default path
- Disclaimer absence treated as blocking defect

### Observability

**Logging:**
- Log level: INFO for evaluation results
- Structured output with indicator match details

## Notes

**Design Decisions:**
- Named complexity indicators (not vague categories) for reliable trigger matching
- Self-help path exists to avoid over-escalation of simple situations
- Cost constraints acknowledged but never suppress warranted referrals
- Jurisdiction-specific questions always escalate to professional referral

**Safety Constraints:**
- Educational guidance ONLY — prominent disclaimer required
- Framework must never confirm legal compliance
- Cost concerns don't override professional referral when warranted
- No commercially-linked attorney referral services
